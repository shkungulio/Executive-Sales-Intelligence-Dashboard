# Executive Sales Intelligence Dashboard

Transforming Excel financial data into an interactive **Power BI executive report** with dynamic visualizations and business insights.

## Project Overview

Organizations frequently store operational and financial data in **Excel spreadsheets**, but extracting meaningful insights from raw tables can be time-consuming and inefficient. This project demonstrates how to transform an Excel dataset into an **interactive Power BI report** that highlights key performance metrics and enables decision-makers to quickly identify trends and opportunities.

Using **Power BI Desktop**, the Excel dataset is imported, transformed, modeled, and visualized into a professional business intelligence dashboard. The resulting report provides an **executive summary of financial performance**, allowing stakeholders to explore profits, product segments, and regional performance through interactive visualizations.

The workflow follows common BI practices including **data preparation, modeling, visualization, and report publishing**.

---

## Business Questions

The report answers key strategic questions such as:

* Which **month and year generated the highest profit**?
* Which **country or region contributes the most revenue**?
* Which **product segments are the most profitable**?
* How do sales and profit vary across **time and market segments**?

These insights help leadership identify areas for **investment, expansion, and optimization**.
